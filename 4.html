option 활용법

* 색상

option 태그를 활용하여 색상을 선택할 수 있는 컨트롤을 만들었다. 개인정보가 아니라서 method=get 유지.
select로 지정한 name와 option의 value가 정보로써 보내지기 때문에,option 태그를 묶어주는 select가 필수이다. 예시로 name=color이고 value=red 인 빨강을 선택하면 color=red 라는 값으로 보내짐.

* 색상2(다중선택)

다중 선택은 간단하게 select 뒤에 multiple을 추가해주면 끝.

위 단일선택 박스는 radio 태그로 만들었다.
radio태그는 은유적인 표현으로 옛날 라디오는 한 버튼을 누르면 다른 버튼은 팝업되었다고..
그런 현상에서 이름을 빌렸기 때문에 이 태그는 한 컨트롤을 누르면 다른 컨트롤은 누를 수 없다. 다중선택이 불가능하다는 뜻.
name값과 value값을 지정하여 어떤 정보가 전송되는지 확실히 드러날 수 있도록 하자!기본 디폴트값을 지정하기 위해 파랑 값에 checked를 추가해주었다.

2-3. input button, reset, hidden

button, reset

기존 input text 컨트롤에 submit과 button, reset을 추가해주었다. submit과 button은 value 값을 따로 지정하여 사용자 가시성을 높였다.

hidden

위 컨트롤은 hidden 태그를 사용하여 만들었다. 겉으로 보았을때는 컨트롤안에 값을 전송하는 것처럼 보이지만, 코드 속 value를 hide라는 name으로 지정하여 함께 전송한다.
어떤 정보를 숨긴채로 보내야 할 때 사용한다고 했다.
